# Interfaz de Administración – Sistema de Librería

## Objetivo General
Diseñar e implementar una interfaz de administración que permita gestionar usuarios, libros y el préstamo de libros mediante operaciones CRUD, facilitando el control básico de una librería.

---

## Justificación
Las interfaces administrativas son fundamentales en cualquier sistema de información, ya que permiten a los usuarios internos (como un recepcionista) gestionar datos de forma organizada y eficiente.

En esta actividad desarrollaremos una interfaz enfocada en la gestión de usuarios, libros y préstamos, aplicando operaciones CRUD y entendiendo cómo se integran estos módulos dentro de un mismo sistema. 
Esto fortalece habilidades en construcción de interfaces, manejo de datos y lógica básica de negocio en entornos reales.

---

## Objetivos Específicos

1. Desarrollar una interfaz de administración que permita:
   - Registrar, listar, editar y eliminar usuarios  
   - Registrar, listar, editar y eliminar libros  

2. Implementar un módulo de gestión de préstamos desde la interfaz:
   - Seleccionar un usuario  
   - Seleccionar un libro  
   - Registrar el préstamo  

3. Visualizar la información de manera clara en la interfaz:
   - Listado de usuarios  
   - Listado de libros  
   - Listado de préstamos realizados  

4. Aplicar operaciones CRUD en todos los módulos:
   - Usuarios  
   - Libros  
   - Préstamos  

5. Mantener una estructura organizada en la interfaz:
   - Formularios claros  
   - Tablas de datos bien presentadas  
   - Navegación simple entre módulos
